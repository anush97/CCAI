 Ordered list of step-by-step execution instructions to accomplish target goal. Specify instructions using unordered markdown list syntax. Instructions may be nested to specify substeps. Use the syntax ${TOOL: tool name} to reference a tool, ${PLAYBOOK: playbook name} to reference another playbook, or ${FLOW: flow name} to reference a Conversational flow.
